\section{Simulation stages}

The simulation application structure is the following:

\begin{enumerate}
  \item \textmark{Parameter parsing}: Command line input parameters are parsed.
  \item \textmark{Simulation initialization}: Initial simulation state is 
        loaded from a file and data structures are initialized.
  \item \textmark{Time steps processing}: For the specified number of time 
        steps the new state for each particle is computed.
  \item \textmark{Results storage}: Final simulation state is saved into a file.
\end{enumerate}
